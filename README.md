# libtb

Go library for sending events to [Tinybird](https://www.tinybird.co/). This is a fork of [libhoney-go](https://godoc.org/github.com/honeycombio/libhoney-go) library.

## Installation:

```
go get -v github.com/ygnuss/libtb-go
```
